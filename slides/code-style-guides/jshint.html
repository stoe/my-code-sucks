JSHint

A JavaScript Code Quality Tool

  • Pros
  • well documented
  • configuration file
    my .jshintrc for this project
  • broad IDE support
    JetBrains, SublimeText, Atom.io, etc.
  • out of the box support for libraries
    jQuery, node, etc.
  • basic ES6 support
  • Cons
  • 2 sets of rules
    enforcing vs relaxing
  • no custom rules
  • no code style support
reference jshint.com
